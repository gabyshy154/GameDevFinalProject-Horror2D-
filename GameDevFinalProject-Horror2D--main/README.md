# GameDevFinalProject-Horror2D

## Project Overview
This project is a 2D top-down psychological horror game developed using the Godot Engine. The game focuses on suspense-driven gameplay where the player explores a mysterious abandoned village while avoiding a hostile entity.

The core experience emphasizes stealth, exploration, and environmental storytelling, where light serves both as a survival tool and a potential threat.

---

## Core Features
- Flashlight-based visibility system with limited functionality  
- Stealth-oriented gameplay with hiding mechanics  
- Sound-based detection system influencing enemy behavior  
- Puzzle-solving elements integrated into the environment  
- Evidence collection system for narrative progression  
- Enemy AI with reactive behavior to player actions  

---

## Controls
| Action        | Key                |
|---------------|-------------------|
| Move          | WASD / Arrow Keys |
| Run           | Shift             |
| Interact      | E                 |
| Flashlight    | F                 |
| Hide          | E (near objects)  |
